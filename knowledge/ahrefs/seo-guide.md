# Ahrefs SEO Complete Guide

## Overview

Ahrefs is an all-in-one SEO toolset for backlink analysis, keyword research, competitor analysis, rank tracking, and site auditing.

## Key Metrics

### Domain Rating (DR)
- Scale: 0-100
- Measures: Backlink profile strength
- Higher = More authoritative domain

### URL Rating (UR)
- Scale: 0-100
- Measures: Individual page backlink strength
- Considers internal and external links

### Traffic Value
- Estimated value of organic traffic
- Based on equivalent PPC cost
- Useful for ROI calculations

### Referring Domains
- Number of unique domains linking to you
- More important than total backlinks
- Quality > Quantity

- Domain Rating

### Backlinks Report

**Analyzing Backlinks**
1. Filter by DR (focus on high-quality)
2. Check dofollow vs nofollow ratio
3. Identify link patterns
4. Find broken backlinks
5. Discover lost links

**Backlink Filters**
- Link type: Dofollow, Nofollow, UGC, Sponsored
- Platform: Blogs, News, Wiki, etc.
- Language
- One link per domain
- New/Lost links

**Anchor Text Analysis**
- Branded anchors (company name)
- Naked URLs
- Generic (click here, learn more)
- Keyword-rich (target keywords)
- Goal: Natural, diverse anchor profile

### Referring Domains

**Quality Indicators**
- High DR sites
- Relevant to your niche
- Real websites (not PBNs)
- Editorial links
- Contextual placement

**Toxic Link Identification**
- Extremely low DR sites
- Foreign language spam
- Thin/duplicate content sites
- Link farms
- Irrelevant directories

## Keywords Explorer

### Keyword Research Process

**1. Seed Keywords**
Enter broad topics to generate ideas

**2. Keyword Ideas Reports**
- **Phrase match**: Contains exact phrase
- **Having same terms**: Contains all terms
- **Also rank for**: Related keywords top pages rank for
- **Search suggestions**: Autocomplete data
- **Newly discovered**: Recently found keywords
- **Questions**: Question-based queries

### Keyword Metrics

**Keyword Difficulty (KD)**
| KD Score | Difficulty | Links Needed |
|----------|------------|--------------|
| 0-10 | Easy | 0-10 |
| 11-30 | Medium | 10-36 |
| 31-50 | Hard | 36-75 |
| 51-70 | Very Hard | 75-200 |
| 71-100 | Extremely Hard | 200+ |

**Search Volume**
- Monthly average
- Location-specific
- Historical trends

**Clicks**
- Estimated clicks from search
- Accounts for SERP features
- Clicks/Search ratio important

**Click Distribution**
- Organic clicks %
- Paid clicks %
- No clicks (zero-click searches)

### SERP Analysis

**For Each Keyword**
- Top 10 ranking pages
- DR of ranking sites
- Backlinks to ranking pages
- Traffic to ranking pages
- SERP features present

**SERP Features**
- Featured snippets
- People also ask
- Image packs
- Video results
- Local packs
- Knowledge panels

### Content Gap Analysis

**Process**
1. Enter your site
2. Enter competitors (up to 10)
3. View keywords they rank for, you don't
4. Filter by volume, KD, intent
5. Prioritize and create content

## Site Audit

### Running an Audit

**Settings**
- Crawl limit (pages)
- Crawl speed
- User agent
- JavaScript rendering
- Robots.txt respect

### Health Score

Percentage based on:
- Critical errors
- Warnings
- Notices

### Issue Categories

**Technical Issues**
- 4xx/5xx errors
- Redirect chains
- Canonical issues
- Orphan pages
- Slow pages

**Content Issues**
- Duplicate content
- Thin content
- Missing H1s
- Multiple H1s
- Missing meta descriptions

**On-Page Issues**
- Missing title tags
- Title too long/short
- Missing meta descriptions
- Keyword stuffing

**Indexability Issues**
- Noindex pages in sitemap
- Blocked by robots.txt
- Missing in sitemap

### Prioritizing Fixes

**Critical (Fix Immediately)**
- 5xx server errors
- Broken redirects
- Orphan pages with traffic
- Security issues

**High Priority**
- 4xx errors
- Duplicate content
- Missing meta data
- Slow pages

**Medium Priority**
- Redirect chains
- Missing alt text
- Mixed content

## Link Building

### Finding Opportunities

**Broken Link Building**
1. Site Explorer > Backlinks
2. Filter: 404 errors
3. Find relevant broken pages
4. Create replacement content
5. Reach out for link placement

**Skyscraper Technique**
1. Find popular content in niche
2. Create something better
3. Reach out to linkers
4. Offer your improved resource

**Guest Posting**
1. Content Explorer search: "write for us" + niche
2. Filter by DR
3. Check editorial guidelines
4. Pitch unique topics

**Resource Page Building**
1. Search: "resources" + niche
2. Find relevant resource pages
3. Suggest your content addition
4. Provide value proposition
