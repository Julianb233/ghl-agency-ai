# PPC Advertising Guide

## Overview

Pay-Per-Click (PPC) advertising is a digital advertising model where advertisers pay each time their ad is clicked. It provides immediate visibility and measurable results.

## PPC Platforms

### Google Ads
- Largest search network
- Display network (2M+ sites)
- YouTube advertising
- Shopping campaigns

### Microsoft Ads (Bing)
- 33% US desktop search market
- Often lower CPCs
- LinkedIn profile targeting
- Older, higher-income audience

### Facebook/Meta Ads
- 2.9B monthly active users
- Advanced targeting options
- Visual ad formats
- Instagram included

### LinkedIn Ads
- B2B focused
- Professional targeting
- Higher CPCs, higher value leads
- Sponsored content, InMail

### Twitter/X Ads
- Real-time targeting
- Trending topics
- Event targeting
- Good for brand awareness

## Keyword Strategy

### Match Types (Google Ads)

| Type | Symbol | Example | Matches |
|------|--------|---------|---------|
| Broad | none | running shoes | jogging sneakers, athletic footwear |
| Phrase | "..." | "running shoes" | best running shoes, running shoes sale |
| Exact | [...] | [running shoes] | running shoes (close variants) |

### Keyword Research Process
```
1. Brainstorm seed keywords
2. Use keyword tools:
   - Google Keyword Planner
   - SEMrush
   - Ahrefs
3. Analyze search volume and competition
4. Group by intent and theme
5. Identify negative keywords
6. Build keyword list
```

### Negative Keywords

**Purpose**: Prevent ads showing for irrelevant searches

**Common Negatives**
```
Unless applicable:
- free, cheap, discount
- jobs, careers, salary
- how to, tutorial, DIY
- reviews, comparison
- used, second hand
```

**Negative Match Types**
- Broad negative: -cheap (blocks any search with "cheap")
- Phrase negative: -"cheap shoes"
- Exact negative: -[cheap shoes]

## Ad Creation

### Search Ad Components

**Headlines** (up to 15)
```
Characters: 30 each
Tips:
- Include keyword
- Unique value proposition
- Call to action
- Numbers and stats
```

**Descriptions** (up to 4)
```
Characters: 90 each
Tips:
- Expand on benefits
- Address pain points
- Include CTA
- Use social proof
```

**Display URL Path**
```
example.com/Running/Shoes
Use relevant keywords
```

### Ad Copy Best Practices
```
✓ Include target keyword in headline
✓ Highlight unique value proposition
✓ Use numbers and statistics
✓ Create urgency (limited time)
✓ Include clear call to action
✓ Address customer pain points
✓ A/B test variations

✗ Don't use excessive punctuation!!!
✗ Don't make false claims
✗ Don't use ALL CAPS
✗ Don't ignore ad policies
```

## Bidding Strategies

### Manual Bidding
- Manual CPC: Set max bid per keyword
- Enhanced CPC: Auto-adjust for conversions

### Automated Bidding

| Strategy | Goal | Best For |
|----------|------|----------|
| Maximize Clicks | Volume | Traffic focus |
| Maximize Conversions | Conversion count | Lead gen |
| Target CPA | Cost control | Fixed budgets |
| Target ROAS | Revenue | E-commerce |
| Target Impression Share | Visibility | Brand |

### Bidding Tips
```
1. Start with manual to learn
2. Gather 30+ conversions before automated
3. Set realistic targets based on data
4. Allow 2 weeks learning period
5. Don't make frequent changes
```

### Key Metrics

**Performance**
```
CTR (Click-Through Rate)
- Formula: Clicks / Impressions
- Search benchmark: 2-5%
- Display benchmark: 0.5-1%

CPC (Cost Per Click)
- Varies by industry
- Track trend over time

Conversion Rate
- Conversions / Clicks
- Benchmark: 2-5%

CPA (Cost Per Acquisition)
- Cost / Conversions
- Target: Profitable for business

ROAS (Return on Ad Spend)
- Revenue / Ad Spend
- Target: Usually 3:1 or higher
```

**Quality**
```
Quality Score (1-10):
- Expected CTR
- Ad relevance
- Landing page experience

Impact on CPC:
QS 10: -50% CPC
QS 7: Baseline
QS 4: +25% CPC
QS 1: +400% CPC
```

## Budget Management

### Budget Allocation

**By Campaign Performance**
```
High ROAS campaigns: Increase budget
Profitable but scaling: Gradual increase
Breaking even: Optimize before scaling
Underperforming: Reduce/pause
```

**By Funnel Stage**
```
Typical split:
- Brand: 10-20%
- Top of funnel: 20-30%
- Middle of funnel: 30-40%
- Bottom of funnel: 20-30%
```

### Forecasting
```
Budget planning:
1. Determine target CPA
2. Set conversion goals
3. Calculate: Budget = Target Conversions × Target CPA
4. Adjust for seasonality
5. Build in testing budget (10-20%)
```

## Common PPC Mistakes

### Strategy
- No clear goals
- Wrong campaign type
- Ignoring search intent
- Not enough budget for learning

### Keywords
- Too few keywords
- Too broad targeting
- Ignoring negative keywords
- Not using match types properly

### Ads
- No A/B testing
- Poor message match
- Weak CTAs
- Not using extensions

### Optimization
- Making changes too frequently
- Ignoring data
- Not tracking conversions
- Set and forget mentality

## PPC + SEO Integration

### Combined Strategy
```
1. Use PPC for keyword testing
2. Target high-competition keywords with PPC
3. Build SEO for long-term keywords
4. Dominate SERP with both organic + paid
5. Use PPC data to inform SEO content
6. Retarget organic visitors with PPC
```

### Data Sharing
- High CTR keywords → SEO content focus
- Converting search terms → Content opportunities
- Landing page winners → Template for SEO pages
- Audience insights → Content personalization
